## [2025-12-25]

### ✨ 新增功能 (Features)
- **应用列表视觉升级**:
  - **磨砂玻璃卡片**: 全新设计应用卡片，以应用背景图作为全卡片背景，叠加高斯模糊与径向渐变，打造现代化的视差与通透感。
  - **风格化按钮**: 下载按钮升级为半透明磨砂风格（Glassmorphism），适配深色/浅色模式，悬停交互更加细腻。

### ⚡ 优化 (Optimizations)
- **后台管理移动端体验**:
  - **友情链接管理**: 优化移动端表格布局，隐藏非核心列（URL），紧凑化图标与状态列，确保操作按钮在有限屏幕内完整显示。
  - **更新日志管理**: 优化操作列最小宽度，解决移动端删除按钮显示不全的问题。
  - **全局弹窗适配**: 统一优化所有管理页面的新增/编辑弹窗，在移动端自动调整为 90% 宽度并将表单标签改为顶部对齐，提升输入体验。
- **UI/UX**:
  - **首页交互**: 修复公告分类下拉菜单在页面滚动时被顶部栏遮挡的问题（优化 z-index 层级至 3000 并调整挂载逻辑）。
  - **视觉修复**: 修复后台管理中 SiteCards 的蓝色 (`bg-blue`) 与紫色 (`bg-purple`) 标签颜色渲染异常的问题。

## [2025-12-24-1]

### ✨ 新增功能 (Features)
- **后台管理移动端适配**:
  - 全面适配后台管理所有模块（首页配置、友链、群聊、应用、公告、环境变量、日志等）的移动端显示。
  - **响应式弹窗**: 编辑/新增对话框在移动端自动调整为全宽或 90% 宽度，表单标签自动切换为顶部对齐。
  - **响应式表格**: 移动端自动隐藏非核心列（如 URL、Key、ID 等），保留关键信息，优化阅读体验。
  - **响应式工具栏**: 顶部筛选和操作按钮组在移动端支持自动换行，防止溢出。

### ⚡ 优化 (Optimizations)
- **首页布局**:
  - 增加首页卡片之间的垂直间距，优化视觉呼吸感，防止内容重叠。
- **首页卡片配置**:
  - 优化表格列宽，将排序权重的排序图标调整至列头最右侧，避免遮挡文字。
- **访客日志**:
  - 在移动端视图中取消对 IP 和设备信息的隐藏，确保关键审计信息的完整性。

### 🐛 修复 (Fixes)
- **HomeView**: 修复首页组件中存在的重复根标签导致的语法错误。
- **TypeScript**: 修复后台管理组件中因缺少 `isMobile` 定义及未使用变量导致的编译报错。

## [2025-12-23]

### ✨ 新增功能 (Features)
- **访客日志管理**:
  - 新增按「地区」和「设备」筛选访客记录的功能（支持下拉选择数据库中已有的记录）。
  - 新增访客记录的批量删除功能。
- **系统日志**:
  - 后台管理新增「系统日志」模块，支持查看系统操作审计日志。
  - 支持系统日志的批量删除及「一键清空」功能。
- **公告系统升级**:
  - 首页公告栏新增多分类切换功能，支持展示不同分类的公告。
  - 实现公告展示优先级逻辑：优先展示分类为「公告」的最新内容，其他分类内容可通过右上角切换按钮查看。
  - 后台 API 更新，支持返回公告分类信息。

### ⚡ 优化 (Optimizations)
- **应用列表交互**:
  - 优化应用卡片交互：取消卡片整体点击跳转，防止误触。
  - 优化下载按钮样式：改为全宽（Full-width）实心按钮，提升视觉显著性和操作便利性。
- **移动端体验**:
  - 优化监控卡片（MonitorCard）在移动端的点击效果，移除默认的灰色高亮背景块。
- **UI/UX**:
  - 优化首页公告栏切换按钮图标，使用「更多（More）」图标替代「排序（Sort）」图标，更符合交互语义。

### 🐛 修复 (Fixes)
- **图表适配**: 修复访客统计图表在 Tab 切换或容器尺寸变化时无法自动重新调整大小的问题（通过 ResizeObserver 监听容器变化）。
- **故障记录交互**: 修复故障记录展开/折叠逻辑，实现点击卡片外部自动折叠，且同一时间仅允许展开一个监控项的故障记录。
- **构建错误**: 移除 `HomeView.vue` 中未使用的 `loadPublic` 函数及相关变量，修复 TypeScript 编译报错（TS6133）。
- **权限问题**: 修复前端调用删除访客接口时未携带 Authorization Token 导致鉴权失败的问题。
